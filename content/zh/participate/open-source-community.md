# 開源社群、攤位及議程軌

您的社群只要與開放原始碼、開源精神或開放協作相關，都可以是 COSCUP 開源社群的一員。COSCUP 在乎的不只是程式碼，也包含開源精神在設計、教育、政策、文化、資料、硬體、社群治理與公共參與等領域的實踐。

COSCUP 發起時，就是由一群來自開源社群的夥伴共同籌辦。近年大會持續提高開源社群參與程度、降低參與門檻，讓社群可以透過議程軌、社群攤位、佈告欄、BoF 與周邊活動一起打造年度開源人年會。

## 參與框架

COSCUP 與許多研討會不同之處，在於議程籌備形式。每一個議程軌通常由開源社群自行籌辦與排定，COSCUP 籌備團隊則負責協調場地、時間、公告、整體會務與必要支援。

開源社群可以在 COSCUP 舉辦的活動不只議題分享，也可以包含：

- 工作坊。
- 社群內部會議。
- 專案貢獻者聚會。
- 社群治理或政策制定討論。
- 新手教學或 onboarding。
- 社群自己的年會、聚會或成果展示。
- 與海外社群共同舉辦的交流活動。
- 申請社群攤位，向會眾介紹社群、專案、活動與參與方式。

社群參與籌備 COSCUP、申請議程軌與申請社群攤位都是免費的。COSCUP 希望降低社群參與門檻，讓更多開源社群能把自己的成果、議題與參與入口帶到會場。

## 社群議程軌

如果您有主辦議程軌，請在活動前確認以下事項。

注意事項：

- 講者是否已收到演講時間、教室位置、議程長度，以及該議程先前討論決定的語言資訊。
- 若議程需要翻譯，請確認是否會有翻譯支援，或是否需要講者自行使用翻譯軟體。
- 講者是否了解錄影、投影片公開與授權安排。COSCUP 預設使用 CC BY 4.0 釋出。
- 議程軌主持人、開場、換場與時間控管是否有人負責。
- 教室門口是否需要張貼當日議程列表。
- 是否需要社群志工協助報到、引導或現場 Q&A。
- 是否已利用社群佈告欄、社群媒體或電子佈告欄宣傳議程。
- 教室設備清單會在活動前由議程組通知。
- 現場不提供 WiFi，若議程、展示或遠端連線需要網路，請自行準備可用的網路方案。

COSCUP 是對所有人開放的活動，嚴禁以售票或預約的方式控制議程參與人數。即使教室已滿，也可以打開大門，讓教室外的會眾圍觀或旁聽。

主持人仍需進行基本秩序管理。若教室內聊天或噪音影響講者與其他會眾，可以禮貌提醒降低音量，或請討論者先至教室外討論後再回到教室。

## 議程軌主持工作介紹

議程軌主持人是社群與會眾之間的重要橋樑，工作通常包含：

- 在每場議程開始前介紹講者與議題。
- 控制時間，提醒講者剩餘時間與換場節奏。
- 協助現場提問、線上提問或會後交流。
- 遇到設備、錄影、滿場、噪音或突發狀況時聯絡大會志工，並且知道當天可用的聯絡管道。
- 確認講者是否願意公開投影片、是否需要補充連結。
- 讓議程軌維持友善、尊重與符合 CoC 的討論環境。

您可以自行決定要不要製作時間牌，例如「5 分鐘」、「3 分鐘」等提醒牌；若議程軌有海外參與者，也建議製作雙語版。

## 議程教室門口裝飾

建議社群自行準備議程教室門口的裝飾與當日議程表，讓會眾經過教室時可以快速知道這間教室的主題、時間與接下來的議程。最簡單的方式是印出紙張張貼在教室門口或附近看板。

建議格式：

- 標題：`教室編號 (日期 Day) - Agenda`，例如 `TR 515 (08/08 週六) - Agenda`、`TR 515 (08/09 週日) - Agenda`
- 議程軌名稱：例如 `XXX Open Source Session Track`
- 日期：例如 `08/08（週六）` 或 `08/09（週日）`
- 教室編號：例如 `TR 515`
- 社群或議程軌名稱
- 當日議程表

範例標題：

```text
TR 515 (08/10 Sunday) - Agenda
XXX Open Source Session Track
```

## 設定社群攤位

COSCUP 兩日活動會聚集大量參與者，社群攤位是招募夥伴、倡議理念、展示專案與宣傳平時聚會的有效方式。近年大會也常安排大地遊戲或集章活動，吸引會眾拜訪各攤位，增加交流機會。

### 場佈與場撤時間

- 場佈時間：08/07（五）下午 2 點之後開放入場場佈。
- 場撤時間：08/09（日）下午 4 點。

大型電器（瞬間功率超過 330 W）未經許可不得使用。物流與臨停卸貨方式會在事前聯絡信中告知。

攤位可以準備：

- 社群簡介與參與方式。
- 專案展示、Demo 或 QRCode。
- 新手入門文件、貢獻指南或活動日曆。
- 貼紙、傳單、紀念品或小互動。
- 中文和英文的材料或文字介紹，讓海外參與者也能認識你們。
- 可回答「我今天可以怎麼開始參與？」的簡短說法。

  社群攤位也可以帶一些貼紙到現場交換，也可以放到貼紙桌上，讓更多參與者自由取用與認識社群。

### 攤位識別與贈品

- 避免擺出含非開源社群 logo 之贈品或貼紙。
- 若有相關貼紙，可移至貼紙交換區。
- 由大會統一安排攤位位置，可能與其他社群攤位併桌。
- 易拉展可放置攤位桌左右兩側，不可影響旁邊攤位。
- 不可使用背版。
- 不得為社群以外的贊助商宣傳，內含廠商 LOGO 的易拉展、壓克力立牌、文宣品均不得擺設。

  ### 攤位裝飾注意事項

- **禁止用膠帶將海報黏貼到牆壁**，請使用腳架或其他不破壞場地的展示方式。
- 如果確實需要黏貼，請勿預設現場人員會接受，請盡可能事先在信中協調與告知。
- 攤位海報禁止包含商業內容，或代表商業公司介紹產品、推廣曝光。
- 若是幫友好社群互相介紹、發放貼紙或分享活動資訊，沒有問題。
- 現場不提供 WiFi，若攤位展示、Demo、集章或收集表單需要網路，請自行準備可用的網路方案。

### 發放食物與物品

若攤位會發放食物或其他可食用物品，敬請使用中文與英文標示以下資訊：

- 葷食或素食
- 含蛋
- 含奶
- 有無麩質
- 牛／豬／羊

:::info{title="攤位提供設備"}

- 通常會提供桌子與椅子，尺寸與數量依當年度公告為準。
- 可能提供 110V 電力插座。大型電器（瞬間功率超過 330 W）未經許可不得使用。
- 建議自行攜帶延長線、轉接頭、立牌、桌巾、固定桌面展示用物品與備用行動電源；請勿將膠帶黏貼在牆面。

:::

## 大地遊戲與集章

若大會安排社群攤位集章，攤位夥伴通常需要使用具有相機功能且可連網的裝置，開啟大會提供的攤位蓋章連結，掃描會眾 OPass QRCode。當畫面顯示蓋章成功，會眾即取得集章點數或任務進度。

請事前檢查您的 Email，我們會將攤位 host 專用的掃描 QRCode 程式安裝方式寄送給您。

## 社群佈告欄

活動場地各樓層可能設有 COSCUP 為社群規劃的專屬佈告欄。無論您的社群是否有攤位，都可以利用佈告欄張貼個人開源專案、社群簡介、活動預告、年會時間表或其他與開源相關的宣傳資訊。

傳單尺寸建議：

- A4：一般傳單，適合放標題、介紹、活動資訊與 QRCode。
- A5：適合社群簡介、專案介紹或折頁文宣。
- A6：適合活動宣傳、問卷、短期 QRCode 或貼紙交換資訊。

:::warning{title="張貼限制"}

除大會提供的看板外，請不要在場地其他地方張貼文件。

:::

## BoF 與周邊活動

我們鼓勵社群利用 COSCUP 活動前後舉辦自己的社群聚會、開發者會議、貢獻者晚餐或跨社群交流。這類活動可以透過 BoF 或大會提供的公告管道讓更多參與者知道。

## 尋求協助

社群夥伴若有任何問題，可以先詢問社群負責人或聯絡人。若需要大會協助，請聯繫對應組別；議程軌相關問題可先聯繫議程組：program [at] coscup.org。
